# SQL-Skill-Demo
Integrated Hospital Management System

Overview
Designed and implemented a normalized healthcare database using MySQL to support patient management, staff administration, medication tracking, and departmental reporting.

Technologies
- MySQL
- SQL
- EER Modeling
- Database Normalization (1NF, 2NF, 3NF)
- MySQL Workbench

Features
- Relational database design
- Supertype/subtype implementation
- Weak entity implementation
- Foreign key constraints
- Healthcare reporting queries
- Department workload analysis
- Medication tracking

Key Skills Demonstrated
- SQL Development
- Database Design
- Data Modeling
- Query Optimization
- Healthcare Data Management
- Data Analysis
